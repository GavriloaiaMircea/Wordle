# Wordle WPF Application

This repository contains a Wordle game application built using Windows Presentation Foundation (WPF) and C#. The application allows users to play the Wordle game with words of varying lengths (3, 4, 5, and 6 letters).

## Features

- Play Wordle with 3, 4, 5, or 6 letter words.
- Simple and intuitive user interface.
- Instructions on how to play the game.
